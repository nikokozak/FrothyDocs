---
title: "Availability"
description: "Current Frothy availability and pre-alpha tester access."
---

Frothy is not publicly installable yet. It is currently available only to
pre-alpha testers, so public install instructions are intentionally not posted
here.

If you are interested in testing Frothy, contact Nikolai at
`nkozak [at] nyu [dot] edu`.

The product is `Frothy`, but the installed CLI command remains `froth` during
the current transition. That naming split is deliberate and will be documented
again when public installation opens.
